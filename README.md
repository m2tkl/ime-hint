# IME Hint

IME Hint is a lightweight floating indicator that shows your IME On/Off state at a glance.

## What It Does
- Prevents mistyping after pauses by keeping your IME state visible while you work
- Provides a subtle on-screen indicator so you can confirm input mode before typing

## Features
- Badge or edge bar display modes (top/left/right/bottom)
- Customizable badge size, colors, and background opacity
- Always-on-top floating window
- Drag to reposition; position is persisted
- Separate settings window (right-click to open)

## Status
- macOS: working
- Windows: planned
