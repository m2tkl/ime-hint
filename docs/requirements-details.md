# Detailed Requirements v2

## Display Policy (Current Implementation)
- Indicator is always visible.
- IME On/Off is expressed via color.

## Position
- Default: fixed position (top-right).
- User can drag to reposition; position is persisted.

## Display Mode
- Badge: fills the window; semi-transparent background.
- Bar: thin bar anchored to an edge (top/left/right/bottom).

## Settings UX
- Settings window opens via right-click on the indicator.
- Settings apply immediately (no explicit Save required).

## Permissions
- macOS Accessibility permission is not required in the current build.

## Open Points
- Windows implementation details and parity
- Auto-launch support
